# Stress Modes

`vico-ground stress` keeps one public route while supporting several internal pressure shapes.

## `check`

Use when you want a quick weakness scan.

- identify the most likely failure point
- do not widen into a full option comparison unless that changes the route

## `tradeoff`

Use when multiple live options exist and the main gap is choosing among them.

- compare options, costs, and irreversibility
- recommend one option or a hold state

## `challenge`

Use when the current proposal or belief needs stronger counterexamples or rebuttals.

- test the current claim set against hidden failure modes
- use this only when stronger pressure meaningfully changes the next decision
